##EXPLICACIÓN 
La diferencia entre git log y git reflog es que git log muestra el historial de commits de las ramas actuales, mientras que git reflog registra todos los movimientos de HEAD, como cambios de rama, commits o merges, incluso si esos commits ya no están en ninguna rama visible.
